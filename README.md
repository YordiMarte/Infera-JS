# Infera.JS

**Infera.JS** is a backend infrastructure platform designed to demonstrate
real-world distributed systems, cloud-native architecture, and DevOps practices.

## Tech Stack

### Languages
- Go
- TypeScript (Node.js)
- Python

### Backend
- NestJS (API Gateway)
- PostgreSQL
- Redis
- Kafka

### Infrastructure
- AWS (EKS, S3)
- Kubernetes
- Terraform
- Docker
- Helm

### CI/CD
- GitHub Actions
- Terraform Pipelines
- Docker Build & Deploy

### Observability
- OpenTelemetry
- Prometheus
- Grafana

## Architecture Overview

- API Gateway (NestJS)
- Internal services (Go)
- Event-driven communication (Kafka)
- Centralized observability and metrics
- Infrastructure as Code (Terraform)
